Privacy Policy — BSN CLANS

Last Updated: 29 November 2025

BSN CLANS is a Discord bot created exclusively for the Blackspire Nation server. This policy explains what data may be collected and how it is handled.

1. Information Collected

  • Discord User ID (only if commands are used)
  • Server ID associated with clan management or bot features
  • Bot configuration and command logs (temporary)

2. Purpose of Data Use

All stored information is used only to provide bot features inside the Blackspire Nation server—such as clan tracking, clan assignments, command functionality, or debugging.

3. Data Sharing

We do not share, sell, or distribute stored information to any third party. Data is only used internally for functionality.

4. Data Removal

You may request deletion of data at any time by contacting staff or opening a support ticket in the Blackspire Nation Discord server.

5. Changes to Policy

This policy may be updated without notice. Continued use of the bot means you accept any changes.

6. Contact

For support, join the Blackspire Nation Discord server (link available on the main site).
